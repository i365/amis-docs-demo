# amis 前端低代码框架 文档 & demo 示例

[amis 官方仓库](https://github.com/baidu/amis)
这是百度公司开源的低代码框架，通过简单的配置就能快速开发后台系统界面，提高开发效率。
无需了解很多前端知识，无需大量的JavaScript编程，通过 JSON 配置，快速开发后台系统。
